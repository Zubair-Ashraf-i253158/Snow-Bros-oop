**Institution:** FAST-NU Islamabad

**Overview**

This is a remake of the classic Snow Bros game, developed in C++ using SFML as part of our OOP project.

The project uses inheritance and polymorphism to manage game logic, including player behavior, enemy AI, and game mechanics.
